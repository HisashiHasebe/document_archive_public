ltdフォルダのファイルにつくt=・・・のURLの有効期限はいくつですか？

有効期限​

トークンの有効期限は15分、もしくはAPIキャッシュ期限になります。

制限に関連するAPIはキャッシュ設定しないで利用する想定ですが、APIキャッシュが設定されている場合はその間リンクが有効となります。

また、現状、有効期間の変更は行えません。

補足​

t=〜のトークンには発行者のグループ権限が含まれています。
APIリクエストを行って発行されたURLを利用する場合には問題ないですが、t=〜のトークンが付いたURLをそのままフロントエンドでリンクとして利用すると意図しない挙動になります。

フロントエンドに張り付けるリンクとして利用する場合は、ファイルマネージャーのFilePathで表示されるAPIドメインを使ってください。
