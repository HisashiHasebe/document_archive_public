File manager

The file manager allows you to upload and manage files such as images, text, and videos for your website. Click [File manager] in the sidebar menu to open it as a pop-up window.

File manager screen​

Summary​

This section gives an overview of all the functions in the file manager window.

1. Upload​

To upload new files, click the [Upload] button. Select as many files as you like from the file dialog. The upload process begins once you have selected the desired file(s). Uploaded images appear as thumbnails, while other file types are displayed as icons.

2. New subfolder​

Click the [New subfolder] button. You will see a dialog for creating a new folder.

The new folder will be listed in the left sidebar.

3. Filter​

Type your input criteria in the "Filter" box. This displays only files with a partial filename match.

4. Display settings​

Click the settings/gear icon to open the custom menu for the file list display.

ItemDescription
SettingsDisplay or hide the file name, timestamp, and file size.
ViewSwitch between list, thumbnail, or compact (icon) view.
Sort bySort by file name, file size, or date and time.
OrderDisplay by ascending or descending order of the "Sort by" criterion.
Thumbnail sizeSelect the icon/thumbnail size (150-500 px).

5. Folder list​

Tree view of the existing folders.
The folders available by default are as follows.

  • KurocoFiles
  • KurocoFiles(With metadata)
  • KurocoFiles(Restricted)

6. File list​

The list of files in the selected folder. The view can be specified under Display settings.

Selected folder actions​

Selecting a folder in the folder list changes the toolbar at the top:

Note: You cannot rename or delete the default KurocoFiles folder.

1. Upload​

Use this button to upload files to the selected folder.

2. New subfolder​

Use this button to create new subfolders in the selected folder.

3. Rename​

Use this button to change the folder name.

4. Delete​

Use this button to delete the selected folder. Clicking [OK] in the dialog box will delete all the files and subfolders contained in the folder.

Selected file actions​

Selecting a file in the file list changes the toolbar at the top. The toolbar controls are slightly different for images and for other file types. The image file toolbar is as follows:

The non-image file toolbar is as follows:

Image files have an additional [Edit] button.

1. Upload​

Use this button to upload files to the selected folder.

2. View​

Zooms in on the selected file. For image files, the content will be displayed. The file icon is displayed for other file types.

3. Download​

Downloads the selected file

4. Edit​

Options for simple editing of the image. After editing, click [Save] to apply the changes made. To undo your edits, click [Reset].

4-1. Resize​

Use this to change the image size. Check the "Keep aspect ratio" box to keep the width-to-height ratio of the original image.

4-2. Crop​

Use this to crop part of the image. Check the "Keep aspect ratio" box to crop an area with the same width-to-height ratio as the original image.

4-3. Rotate​

Use this to rotate the image 90 degrees right or left. To turn the image upside-down, execute the command twice.

4-4. Adjust​

Use this to adjust the following visual properties of the image:

  • Brightness
  • Contrast
  • Saturation
  • Exposure
  • Sepia tone
  • Sharpness

4-5. Presets​

Use this to select one of the following preset filters for the image:

  • Clarity
  • Her Majesty
  • Nostalgia
  • Pinhole
  • Sunrise
  • Vintage

5. Copy​

Copies files to the selected folder. To select a subfolder lower in the hierarchy, click the triangle to the right side of the folder name. Note: files cannot be copied to the same folder.

6. Move​

Moves files to the selected folder. To select a subfolder lower in the hierarchy, click the triangle to the right side of the folder name. Note: files cannot be moved to the same folder.

7. Rename​

Renames the selected file. The default value is the original file name. Click [OK] to save the changes.

8. File path​

Displays the path and URL of the selected file. Click on the link in the dialog box to open the URL.

Please use the API domain when using the file path of the KurocoFiles(Restricted) folder in the front-end. It is intended for the following purposes:

  • URL based on the Admin panel domain: for site administrators to confirm files.
  • URL based on the API domain: for links used in the front-end.

9. Delete​

Deletes the selected file. Click [OK] in the dialog box to execute.

Unzip​

Right-click on the uploaded ZIP file and click "Unzip" to extract the ZIP file in the file manager.
You can upload multiple files at once by extracting the ZIP file within the file manager.

Setting Folder Permissions​

Right-click on a folder you have created (other than the root folder) and click on [Folder ACL] to set permissions on the folder.

Permissions for open folders (KurocoFiles)​

For open folders only write permissions could be setup. Mark groups you want allow access to selected folders and confirm your selection with click on [OK] button.

Permissions for private folders (KurocoFiles(Restricted))​

For private folders write and read permissions could be setup. Mark groups you want allow access to selected folders and confirm your selection with click on [OK] button.

If a user without read permission accesses the target file, it will display "The requested page was not found.".

Permission inheritance​

If you setup permissions for parent folder all it's subfolders will have same permissions. If you need specific permissions for subfolder you can set up it directly. Subfolder permissions have priority on the parent permissions.

Folder Extensions​

External system integration allows you to save files from File Manager to GCS and S3.
Please refer to the following document for details on how to cooperate.

  • Cloud storage integration with Firebase
  • Integrating and storing files with Amazon S3
caution

You can't use both GCS and Amazon S3.
